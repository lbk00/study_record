
> 기획 작품 

- ppt 문제 생성 사이트 or 앱

> 사용 언어

- Python & ( 필요 할 경우 스프링 )

> 기획  목적

- ppt에 있는 텍스트를 인식하여 문제를 자동 생성해주는 프로그램

> 기대 효과

- ppt를 아용해 공부할때 개념과 내용을 연결시켜 공부하는데 도움을 줄 수 있음

> 주요 기능

- 개념부분은 빨간색 형광펜으로 표시하여 텍스트 인식

- 내용부분은 노란색 형관펜으로 표시하여 텍스트 인식

- 개념과 내용 중 하나를 빈칸으로 제공하고 빈칸을 추측할수 있도록 구현
