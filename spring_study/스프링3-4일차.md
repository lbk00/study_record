> ### 테스트코드 작성 요령
> 
> //given  데이터가 이런 상황일 때, 혹은 주어졌을 때
> 
> //when   내가 어떤 작업을 수행하면
> 
> //then   그럼 이런 결과가 나온다.

> 테스트 코드에서는 함수 한글로 적어줘도 무방
 
> ### 스프링 빈을 등록하는 2가지 방법
> - 컴포넌트 스캔과 자동 의존관계 설정  @Autowired // @Controller -> @Service -> @Repository
> - 자바 코드로 직접 스프링 빈 등록하기

> @GetMapping -> 데이터 조회할때
> 
> @PostMapping -> 데이터 등록할때
