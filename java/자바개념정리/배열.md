> ### 배열
>
> 동일한 타입의 데이터를 메모리에 연속적으로 저장한 것

> - 배열을 선언하는 방법
>   
> `int[] arr = {10,20,30}`
>
> `int[] arr = new int[3]`// 3의 크기를 가지고 초기값 0으로 채워진 배열
>
> `int[] arr = new int[] {10,20,30}`
>
> `int[] arr;
>  arr = new int[] {10,20,30}`

> - 2차원 배열
>
> `int[][] arr = new int[3][5]`

> - 배열의 동적할당
>
> `ArrayList<참조타입> arr = new ArrayList<>();`
>
> `arr.add(10)` // 데이터를 추가
>
> `arr.remove(0)` // 인덱스 번호의 원소 제거
